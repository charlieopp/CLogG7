Search Grammar

Simple Search:

  • term - highlights "term" everywhere
  • term1, term2 - highlights both terms everywhere

Logical Search (per line):

  • term1 _& term2 - highlights both terms only in lines containing both
  • term1 _| term2 - highlights both terms in lines containing either
  • _! term - highlights all terms in lines NOT containing "term"
  • _( term1 _& term2 _) - grouping for complex expressions

Examples:

  • ERROR _& Service - "ERROR" and "Service" highlighted only in lines with both
  • _( INFO _| DEBUG _) _& timeout - complex grouping
